# Welcome to NodeTL Documentation

We're excited to launch the official documentation website for **NodeTL** - a powerful visual data mapping and transformation platform.

<!-- truncate -->

## What is NodeTL?

NodeTL is an open-source platform that simplifies data integration through visual interfaces. Whether you're building ETL pipelines, integrating APIs, or automating data workflows, NodeTL provides the tools you need.

## Key Features

- **Visual Data Mapping** - Drag-and-drop interface for field mapping
- **Workflow Automation** - Node-based visual designer
- **AI-Powered** - Generate test data and get mapping suggestions
- **Enterprise Ready** - RBAC, OAuth, and audit logging
